##### 升级说明
	STM32F103VC:
		正常情况下:
		IROM1: 0x8000000/0x40000
		IRAM1: 0x20000000/0xC000

		需要IAP时:
		IROM1: 0x8002000/0xC800
		IRAM1: 0x20000000/0xC000
		增加:
		SCB->VTOR = FLASH_BASE | 0x2000;
	
	STM32F103R8:
		正常情况下:
		IROM1: 0x8000000/0x10000
		IRAM1: 0x20000000/0x5000

		需要IAP时:
		IROM1: 0x8002000/0xC800
		IRAM1: 0x20000000/0x5000
		增加:
		SCB->VTOR = FLASH_BASE | 0x2000;
